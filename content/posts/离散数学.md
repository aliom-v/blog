## 一、计算题

### 1.

**题目**：一个树有 2 个 2 度顶点， 3 个 3 度顶点， 4 个 4 度顶点，其余顶点度数都是 1 ，问这个树有几个 1 度顶点？

**答案**：13个

**解析**：
- 设1度顶点（叶子）有x个
- 树的顶点总数 n = 2 + 3 + 4 + x = 9 + x
- 树的边数 m = n - 1 = 8 + x
- 度数和 = 2×2 + 3×3 + 4×4 + 1×x = 4 + 9 + 16 + x = 29 + x
- 根据握手定理：度数和 = 2×边数
- 29 + x = 2(8 + x) = 16 + 2x
- **x = 13**

```
            90
          /    \
        40      50
       /  \    /  \
     20    20 25   25(原25)
    / \     \  \
   10  10   10  10+15
  / \          |
  5   5       15
```

**各节点深度**：
- 25(原25): 深度2
- 25(新): 深度3，包含10和15
- 20(右): 深度3，是原来的10
- 20(左): 深度3，包含10+10
- 15: 深度4
- 10(20左的右): 深度4
- 10(25内的): 深度4
- 10(20左的左): 深度4

- 25×2 = 50
- 10×3 = 30（20(右)）
- 10×3 = 30（20左的左10）
- 10×3 = 30（20左的右10）
- 15×3 = 45（25内的15）
- 10×4 = 40（25内的10）
- 5×5 = 25
- 5×5 = 25

WPL = 50+30+30+30+45+40+25+25 = **275**

**最佳前缀码**：
- 0(25)：00
- 1(20)：010
- 2(15)：011
- 3(10)：100
- 4(10)：101
- 5(10)：110
- 6(5)：1110
- 7(5)：1111

### 49.

**题目**：用真值表方法判断下列推理是否正确。p→q, r∧s, ¬q => p ∧ s

**答案**：**错**

**解析**：
- 前提1：p→q
- 前提2：r∧s
- 前提3：¬q
- 结论：p∧s

要使前提全真，需要：
- r=T, s=T（由r∧s）
- q=F（由¬q）
- p=F（因为q=F，要使p→q为真，需要p=F）

此时结论p∧s = F∧T = F为假。

前提全真而结论假，所以**推理不正确**。

## 附录：常用逻辑等价式

1. **德摩根律**：
   - ¬(p∧q) ≡ ¬p ∨ ¬q
   - ¬(p∨q) ≡ ¬p ∧ ¬q

2. **蕴含等价**：
   - p→q ≡ ¬p ∨ q

3. **双重否定**：
   - ¬¬p ≡ p

4. **分配律**：
   - p∧(q∨r) ≡ (p∧q) ∨ (p∧r)
   - p∨(q∧r) ≡ (p∨q) ∧ (p∨r)

5. **量词否定**：
   - ¬∀xP(x) ≡ ∃x¬P(x)
   - ¬∃xP(x) ≡ ∀x¬P(x)

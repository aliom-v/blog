# 高等数学复习题及答案

## 一、选择题

**1. 下列函数中为奇函数的是（ A ）**

A. sin x cos³x　　B. sin x / x　　C. (1/2)(eˣ + e⁻ˣ)　　D. f(x²)

**解析**：A选项 f(-x) = sin(-x)cos³(-x) = -sin x · cos³x = -f(x)，是奇函数。B是偶函数，C是偶函数。

---

**2. 下列各组函数中，是相同的函数的是（ B ）**

(A) f(x) = ln x² 和 g(x) = 2ln x　　(B) f(x) = |x| 和 g(x) = √(x²)
(C) f(x) = x 和 g(x) = (√x)²　　(D) f(x) = |x|/x 和 g(x) = 1

**解析**：B选项两函数定义域均为R，且|x| = √(x²)恒成立。A选项定义域不同（ln x²对x≠0，2ln x要求x>0）。

---

**3. 函数 f(x) 在点 x₀ 处连续是 lim(x→x₀) f(x) 存在的（ B ）条件**

A. 必要　　B. 充分　　C. 充分必要　　D. 无关

**解析**：连续⇒极限存在，但极限存在不一定连续（极限可能≠f(x₀)），故连续是充分不必要条件。

---

**4. 函数 f(x) 在点 x₀ 处可导是 lim(x→x₀) f(x) 存在的（ B ）条件**

A. 必要　　B. 充分　　C. 充分必要　　D. 无关

**解析**：可导⇒连续⇒极限存在，故可导是极限存在的充分条件。

---

**5. f(x) = eˣ 在区间(0,1)内满足拉格朗日定理的 ξ 是（ A ）**

A. ln(e-1)　　B. 1/ln(e-1)　　C. e-1　　D. 1/(e-1)

**解析**：由拉格朗日中值定理：e^ξ = (e¹ - e⁰)/(1-0) = e - 1，故 ξ = ln(e-1)。

---

**6. 下列等式不成立的是（ D ）**

A. d/dx ∫ₐᵇ f(x)dx = 0　　B. d/dx ∫f(x)dx = f(x)
C. d/dx ∫ₐˣ² f(t)dt = 2xf(x²)　　D. d²/dx² ∫ₐˣ² f(t)dt = f(x²)

**解析**：D选项，设F(x) = ∫ₐˣ² f(t)dt，则F'(x) = 2xf(x²)，F''(x) = 2f(x²) + 4x²f'(x²) ≠ f(x²)。

---

## 二、计算题

**6. 计算 lim(x→a) (sin x - sin a)/(x - a)**

**答案**：cos a

**解析**：这是 sin x 在 x = a 处的导数定义式，即 (sin x)'|ₓ₌ₐ = cos a。

---

**7. 求极限 lim(x→0) (1 - x/2)^(1/x)**

**答案**：e^(-1/2) = 1/√e

**解析**：ln y = (1/x)ln(1 - x/2) ≈ (1/x)·(-x/2) = -1/2，故 y = e^(-1/2)。

---

**8. 求极限 lim(x→0) (1-x)^(1/x)**

**答案**：e⁻¹ = 1/e

**解析**：ln y = (1/x)ln(1-x) ≈ (1/x)·(-x) = -1，故 y = e⁻¹。

---

**9. lim(x→0) (eˣ - 1)/sin x**

**答案**：1

**解析**：eˣ - 1 ~ x，sin x ~ x（x→0），故原式 = lim(x→0) x/x = 1。

---

**10. lim(x→0) (x - sin x)/x³**

**答案**：1/6

**解析**：泰勒展开 sin x = x - x³/6 + o(x³)，故 x - sin x ~ x³/6，原式 = 1/6。

---

**11. 求曲线 f(x) = (x+1)/(x² - x - 2) 的垂直渐近线**

**答案**：x = 2

**解析**：x² - x - 2 = (x-2)(x+1)，f(x) = 1/(x-2)（x≠-1），x = -1 是可去间断点，x = 2 是垂直渐近线。

---

**12. 求曲线 y = 1/|x| 的水平渐近线和垂直渐近线**

**答案**：水平渐近线 y = 0，垂直渐近线 x = 0

**解析**：lim(x→±∞) 1/|x| = 0，lim(x→0) 1/|x| = +∞。

---

**13. 求微分方程 dy/dx = y/x 的通解**

**答案**：y = Cx（C为任意常数）

**解析**：分离变量 dy/y = dx/x，积分得 ln|y| = ln|x| + C₁，即 y = Cx。

---

**14. 微分方程 dy/dx = y 的通解**

**答案**：y = Ceˣ（C为任意常数）

**解析**：分离变量 dy/y = dx，积分得 ln|y| = x + C₁，即 y = Ceˣ。

---

**15. 设函数 y = y(x) 由方程 e^(x+y) + cos(xy) = 0 确定，求 dy/dx**

**答案**：dy/dx = -(e^(x+y) - y·sin(xy))/(e^(x+y) - x·sin(xy))

**解析**：两边对x求导：e^(x+y)(1 + y') - sin(xy)(y + xy') = 0，整理得 y'。

---

**16. 设 y = 1 + xeʸ，则 dy/dx**

**答案**：dy/dx = eʸ/(1 - xeʸ)

**解析**：两边对x求导：y' = eʸ + xeʸy'，解得 y' = eʸ/(1 - xeʸ)。

---

**17. 求函数 f(x) = (1/2)(eˣ - e⁻ˣ) 的拐点**

**答案**：(0, 0)

**解析**：f''(x) = (1/2)(eˣ - e⁻ˣ) = f(x)，f''(0) = 0 且两侧变号，拐点为(0, 0)。

---

**18. 求函数 y = xe⁻³ˣ 的拐点**

**答案**：(2/3, (2/3)e⁻²)

**解析**：y' = e⁻³ˣ(1 - 3x)，y'' = e⁻³ˣ(9x - 6) = 0，得 x = 2/3。

---

**19. ∫ tan⁴x sec²x dx**

**答案**：(1/5)tan⁵x + C

**解析**：设 u = tan x，du = sec²x dx，原式 = ∫u⁴du = u⁵/5 + C。

---

**20. ∫ tan³x sec²x dx**

**答案**：(1/4)tan⁴x + C

**解析**：设 u = tan x，du = sec²x dx，原式 = ∫u³du = u⁴/4 + C。

---

**21. ∫₋₁¹ x√(1-x²) dx**

**答案**：0

**解析**：被积函数 x√(1-x²) 是奇函数，在对称区间[-1,1]上积分为0。

---

**22. ∫₋₁¹ sin x · √(1+x²) dx**

**答案**：0

**解析**：sin x 是奇函数，√(1+x²) 是偶函数，乘积为奇函数，对称区间积分为0。

---

**23. 求 ∫ arctan x dx**

**答案**：x·arctan x - (1/2)ln(1+x²) + C

**解析**：分部积分，∫arctan x dx = x·arctan x - ∫x/(1+x²)dx = x·arctan x - (1/2)ln(1+x²) + C。

---

**24. 求 ∫ arcsin x dx**

**答案**：x·arcsin x + √(1-x²) + C

**解析**：分部积分，∫arcsin x dx = x·arcsin x - ∫x/√(1-x²)dx = x·arcsin x + √(1-x²) + C。

---

**25. 求 ∫₀⁴ e^√(2x+1) dx**

**答案**：2e³

**解析**：设 t = √(2x+1)，dx = t dt，x:0→4 对应 t:1→3。原式 = ∫₁³ teᵗdt = [(t-1)eᵗ]₁³ = 2e³ - 0 = 2e³。

---

**26. 求 ∫₀⁴ e^√x dx**

**答案**：2(e² + 1)

**解析**：设 t = √x，dx = 2t dt，原式 = 2∫₀² teᵗdt = 2[(t-1)eᵗ]₀² = 2[e² - (-1)] = 2e² + 2。

---

**27. 反常积分 ∫₀^(+∞) e⁻ˣ dx**

**答案**：1

**解析**：∫₀^(+∞) e⁻ˣdx = [-e⁻ˣ]₀^(+∞) = 0 - (-1) = 1。

---

**28. 微分方程 y'' - 2y' + y = 0 的通解**

**答案**：y = (C₁ + C₂x)eˣ

**解析**：特征方程 r² - 2r + 1 = 0，(r-1)² = 0，r = 1（二重根）。

---

**29. 求微分方程 y'' + 6y' + 13y = 0 的通解**

**答案**：y = e⁻³ˣ(C₁cos 2x + C₂sin 2x)

**解析**：特征方程 r² + 6r + 13 = 0，r = -3 ± 2i。

---

**30. 设函数 f(x) = {x², x≤1; ax+b, x>1} 在 x=1 处连续且可导，求 a，b 的值**

**答案**：a = 2，b = -1

**解析**：连续性：1 = a + b；可导性：左导数 2 = 右导数 a，故 a = 2，b = -1。

---

**31. 设 f(x) = {x²sin(1/x), x>0; ax+b, x≤0} 在 x=0 处可导，求 a，b 的值**

**答案**：a = 0，b = 0

**解析**：连续性要求 lim(x→0⁺) x²sin(1/x) = 0 = b；f'₊(0) = lim(h→0⁺) h·sin(1/h) = 0 = a。

---

**32. 求 f(x) = x³ - 3x² - 9x + 5 的极值**

**答案**：极大值 f(-1) = 10，极小值 f(3) = -22

**解析**：f'(x) = 3(x+1)(x-3) = 0，x = -1, 3。f''(-1) < 0（极大），f''(3) > 0（极小）。

---

**33. 求函数 f(x) = 4 + 8x³ - 3x⁴ 的极值**

**答案**：极大值 f(2) = 20

**解析**：f'(x) = 12x²(2-x) = 0，x = 0 或 x = 2。x = 0 处导数不变号（非极值），x = 2 是极大值点。

---

**34. 设 f(x) 由参数方程 {x = ln(1+t²), y = t - arctan t} 确定，求 t=0 处的切线方程及法线方程**

**答案**：切线方程 y = 0，法线方程 x = 0

**解析**：t=0 时 (x,y) = (0,0)。dy/dx = (t²/(1+t²))/(2t/(1+t²)) = t/2，t→0 时斜率为0。

---

**35. {x = ln t, y = t³}，求 d²y/dx²|_{t=1}**

**答案**：9

**解析**：dy/dx = 3t²/(1/t) = 3t³，d²y/dx² = (9t²)/(1/t) = 9t³，t=1 时为9。

---

**36. 求微分方程 dy/dx + y = e⁻ˣ 的通解**

**答案**：y = (x + C)e⁻ˣ

**解析**：一阶线性方程，通解 y = e⁻ˣ(∫e⁻ˣ·eˣdx + C) = (x + C)e⁻ˣ。

---

**37. 求微分方程 xy' + y = xeˣ 的通解**

**答案**：y = ((x-1)eˣ + C)/x

**解析**：化为 y' + y/x = eˣ，通解 y = (1/x)(∫xeˣdx + C) = ((x-1)eˣ + C)/x。

---

**38. 设曲线 xy = 1 与直线 y = x 及 x = 2 所围平面图形为D**

**(1) 求平面图形D的面积**

**答案**：S = 3/2 - ln 2

**解析**：交点(1,1)。S = ∫₁²(x - 1/x)dx = [x²/2 - ln x]₁² = (2 - ln 2) - 1/2 = 3/2 - ln 2。

**(2) 求D绕x轴旋转一周所成旋转体的体积**

**答案**：V = 11π/6

**解析**：V = π∫₁²(x² - 1/x²)dx = π[x³/3 + 1/x]₁² = π[(8/3 + 1/2) - (1/3 + 1)] = 11π/6。

---

**39. 计算由两条抛物线 y² = x, y = x² 所围成的图形的面积及绕x轴旋转一周所成旋转体的体积**

**面积答案**：S = 1/3

**解析**：交点(0,0)和(1,1)。S = ∫₀¹(√x - x²)dx = 2/3 - 1/3 = 1/3。

**体积答案**：V = 3π/10

**解析**：V = π∫₀¹(x - x⁴)dx = π[x²/2 - x⁵/5]₀¹ = π(1/2 - 1/5) = 3π/10。

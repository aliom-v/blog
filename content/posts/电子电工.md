## 一、判断题

**1.** 电位值是相对的（取决于参考点），而电压（两点电位之差）是绝对的，它不随参考点的不同而改变。

**答案：√**


**2.** 电源等效变换只是对于实际电压源和实际电流源有效，理想电压源和理想电流源之间不能进行等效变换。

**答案：√**


**3.** $u=5\sin(30t+30^\circ)\text{V}$ 与 $i=7\sin(30t+10^\circ)\text{A}$ 的相位差为 $30^\circ$。

**答案：×**

**解析**：相位差 $\Delta\phi = \psi_u - \psi_i = 30^\circ - 10^\circ = \mathbf{20^\circ}$


**4.** 硅型材料半导体二极管的导通压降为 $0.2\text{V} \sim 0.3\text{V}$。

**答案：×**

**解析**：硅管导通压降通常为 $0.6\text{V} \sim 0.7\text{V}$；$0.2\text{V} \sim 0.3\text{V}$ 是**锗管**的特征


**5.** 双极性晶体管是电流控制电流源，场效应管是电压控制电流源。

**答案：√**


**6.** 放大电路必须加上合适的直流电源才能正常工作。

**答案：√**


**7.** 基本 RS 触发器是构成其它类型触发器的基本电路。

**答案：√**

**2.** 若某电源开路电压为 $12\text{V}$，短路电流为 $1\text{A}$，则负载能获得的最大功率为：

A. 3W
B. 6W
C. 12W
D. 24W

**答案：A**

**解析**：电源内阻 $R_o = U_{oc} / I_{sc} = 12\text{V} / 1\text{A} = 12\Omega$。最大功率 $P_{max} = U_{oc}^2 / (4R_o) = 144 / 48 = 3\text{W}$

D. $1/Z = 1/Z_1 + 1/Z_2 + \dots + 1/Z_n$

**答案：D**

## 四、计算与分析题

### 1. 直流电路分析（支路电流法）

**题目**：已知电路中 $U_{s1}=70\text{V}, U_{s2}=120\text{V}, R_1=10\Omega, R_2=15\Omega, R_3=12\Omega$。求各支路电流。

**解题过程**：

1. 列 KCL 方程（针对节点 a）：
   $$I_1 + I_2 = I_3 \quad (1)$$

2. 列 KVL 方程（针对左、右两个网孔）：
   - 左回路：$R_1 I_1 + R_3 I_3 = U_{s1} \Rightarrow 10 I_1 + 12 I_3 = 70 \quad (2)$
   - 右回路：$-R_2 I_2 + U_{s2} - R_3 I_3 = 0 \Rightarrow -15 I_2 + 120 - 12 I_3 = 0 \quad (3)$

3. 联立求解：将 (1) 代入 (2) 和 (3)：
   - $10 I_1 + 12(I_1 + I_2) = 70 \Rightarrow 22 I_1 + 12 I_2 = 70$
   - $120 - 15 I_2 - 12(I_1 + I_2) = 0 \Rightarrow 12 I_1 + 27 I_2 = 120$

4. 解得结果：**$I_1 = 1\text{A}, I_2 = 4\text{A}, I_3 = 5\text{A}$**


### 2. 共射放大电路计算

**题目**：设 $V_{CC}=20\text{V}, R_C=4\text{k}\Omega, R_B=400\text{k}\Omega, \beta=40, r_{be}=0.8\text{k}\Omega, U_{BE}$ 忽略。

**解题过程**：

1. 静态分析：
   - $I_{BQ} = V_{CC} / R_B = 20\text{V} / 400\text{k}\Omega = \mathbf{0.05\text{mA}}$
   - $I_{CQ} = \beta I_{BQ} = 40 \times 0.05 = \mathbf{2\text{mA}}$
   - $U_{CEQ} = V_{CC} - I_{CQ} R_C = 20 - 2 \times 4 = \mathbf{12\text{V}}$

2. 动态分析：
   - 当负载 $R_L=4\text{k}\Omega$ 时：
     $A_u = -\beta (R_C // R_L) / r_{be} = -40 \times (4 // 4) / 0.8 = \mathbf{-100}$
   - 当空载时（$R_L$ 开路）：
     $A_u = -\beta R_C / r_{be} = -40 \times 4 / 0.8 = \mathbf{-200}$
   - 阻抗计算：输入电阻 $r_i = R_B // r_{be} \approx 0.8\text{k}\Omega$；输出电阻 $r_o = R_C = 4\text{k}\Omega$


### 3. 集成运放电路分析

**题目**：已知 $R_1=R_2=R_3=R_{f2}=30\text{k}\Omega, R_{f1}=90\text{k}\Omega$。若 $u_{i1}=2\text{V}, u_{i2}=1\text{V}$，求 $u_o$。

**解题过程**：

1. 分析电路结构：电路 $A_1$ 为同相比例运算电路，$A_2$ 为反相加法运算电路

2. 第一级输出 $u_{o1}$：
   $u_{o1} = (1 + R_{f1} / R_1) u_{i1} = (1 + 90/30) \times 2 = \mathbf{8\text{V}}$

3. 第二级输出 $u_{o2}$（即最终输出 $u_o$）：
   $u_{o2} = -(R_{f2} / R_2 \cdot u_{i2} + R_{f2} / R_3 \cdot u_{o1}) = -(30/30 \times 1 + 30/30 \times 8) = \mathbf{-9\text{V}}$


### 4. 逻辑门电路分析

**题目**：根据逻辑图写出表达式并分析功能。

**解题过程**：

1. 写出逻辑表达式：图中输出 $F = P + Q$，其中 $P = AB$（与门），$Q = \overline{A+B} = \overline{A}\overline{B}$（或非门）

2. 组合结果：$F = AB + \overline{A}\overline{B}$

3. 功能分析：该表达式代表的是 **同或门**（XNOR），即当输入 A 和 B 相同时，输出 F 为 1


### 5. JK触发器波形分析

**题目**：JK 触发器初态为 0，$J=K=1$，画出输出波形。

**解题过程**：

1. 触发特性：该 JK 触发器为下降沿触发（CP 信号带小圆圈）

2. 状态规律：由于 $J=K=1$，根据 JK 触发器的特性方程 $Q^{n+1} = \overline{Q^n}$，触发器在每个时钟脉冲的**下降沿**发生翻转

3. 波形结果：Q 端在第 1 个下降沿从 0 跳变到 1，在第 2 个下降沿从 1 跳变到 0，以此类推，输出波形为周期是 CP 两倍的方波
